# AIOffice Project Brain

This is the fresh-summon primer for the clean reset repo.

## What This Repo Is
A clean rebuild of AIOffice focused on one narrow proof slice:
- admin-only self-build V1
- supervised workflow through `architect`
- bounded reviewed `apply/promotion` control

## What This Repo Is Not
- not a migration of donor backlog history
- not proof of a broad UI or control-room
- not proof of later-lane automation or broad orchestration
- not a Standard or subproject pipeline build
